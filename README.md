# 📱 React-Native-Anwar

Selamat datang di repositori utama untuk pengembangan dan pembelajaran **React Native**. Repositori ini digunakan sebagai ruang kerja (workspace) sentral untuk menyimpan berbagai proyek, tugas praktikum, dan eksperimen aplikasi mobile.

---

## 🚀 Teknologi yang Digunakan
- **[React Native](https://reactnative.dev/)** - Framework utama untuk membangun aplikasi mobile lintas platform.
- **[Expo](https://expo.dev/)** - Toolchain yang memudahkan proses development dan build.
- **Node.js & npm** - Lingkungan runtime JavaScript dan package manager.
- **React Navigation** - Untuk pengaturan rute dan navigasi antar layar.

---

## 📂 Struktur Repositori
Repositori ini akan berisi berbagai sub-proyek. Struktur standarnya adalah sebagai berikut:

```text
React-Native-Anwar/
│
├── project-1/          # Contoh folder proyek pertama
├── project-2/          # Contoh folder proyek kedua
│
├── README.md           # Dokumentasi utama repositori ini
└── .gitignore          # File pengecualian Git
```

*(Catatan: Setiap folder proyek akan memiliki file `package.json` dan konfigurasi mandirinya masing-masing).*

## 💻 Cara Menjalankan Proyek
Untuk menjalankan salah satu proyek di dalam repositori ini, ikuti langkah-langkah berikut:

1. **Clone Repositori:**
   ```bash
   git clone https://github.com/anwarrohmadi2006/React-Native-Anwar.git
   cd React-Native-Anwar
   ```
2. **Masuk ke Direktori Proyek Spesifik:**
   Pilih folder proyek yang ingin dijalankan.
   ```bash
   cd nama-folder-proyek
   ```
3. **Instalasi Dependensi:**
   ```bash
   npm install
   ```
4. **Mulai Development Server (Expo):**
   ```bash
   npx expo start
   ```
   > 💡 **Tip:** Gunakan aplikasi **Expo Go** di smartphone Anda untuk memindai kode QR yang muncul di terminal, atau tekan `a` untuk menjalankannya di Android Emulator.
