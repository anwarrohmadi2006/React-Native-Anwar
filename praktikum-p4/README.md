# Praktikum P4 - React Native
Modul 4 Mobile Programming - Implementasi Navigation, FlatList, dan SectionList.

## Fitur
1. **Daftar Mahasiswa (FlatList)**: Layout grid dengan pencarian, filter, pull-to-refresh.
2. **Jadwal Kuliah (SectionList)**: Jadwal dikelompokkan berdasarkan hari dengan sticky headers.
3. **Katalog Produk**: Daftar produk dengan custom column, skeletons, dan filter dinamis.
4. **Navigasi Terintegrasi**: Nested Bottom Tab Navigator & Stack Navigator.

## Setup & Instalasi (Expo SDK 55)
Pastikan Anda sudah menginstall dependencies.

```bash
npm install
npx expo start
```

*Dibangun menggunakan React Native v0.83 dan Expo SDK 55 (New Architecture).*
